O site tem a ideia de uma central de vídeos sobre hardware e notícias do meio, unindo em
um só lugar a comunidade interessada neste campo da tecnologia.

Este site foi produzido para o projeto Recriando a Interface do Netflix,
apenas com fins de estudos. Majoritariamente dos texto e imagens utilizadas no site, foram
salvas do site Techspot (https://www.techspot.com/).

Os ícones utilizados foram retirados do site flaticon (https://www.flaticon.com/), sem fins comerciais.
